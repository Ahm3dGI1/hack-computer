Each project in this repo corresponds to a major step in the computer construction process:
1. **Boolean Logic:** Basic logic gates built from NAND.
2. **Boolean Arithmetic:** Adders, ALU components, and arithmetic chips.
3. **Sequential Logic:** Memory elements like registers, RAM, and counters.
4. **Machine Language:** Assembly programs written in the Hack language.
5. **Computer Architecture:** The complete CPU and memory system.
6. **Assembler:** Translating Hack assembly into binary machine code.
7. **Virtual Machine (VM):** Implementation of the VM translator.
8. **High-Level Language:** Compiler elements for the Jack programming language.
9. **Operating System:** Core system utilities written in Jack.
